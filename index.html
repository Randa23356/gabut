Installation

Terminal

# Clone the repository
git clone https://github.com/yourusername/ftwodev-template.git

# Navigate to project directory
cd ftwodev-template

# Open index.html in your browser
open index.html

That's it! No build process required. Just open the HTML file in your browser.

palette

Customization

tailwind.config.js

// Customize colors
tailwind.config = {
    theme: {
        extend: {
            colors: {
                primary: '#30e87a',
                secondary: '#6b8afd',
                accent: '#ff6b9d',
                // Add your custom colors
                brand: '#ff6b35'
            }
        }
    }
}

Easily customize colors, fonts, and other styles in the Tailwind config.
